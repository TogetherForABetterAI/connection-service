# Connection Service

This service acts as a gateway for authentication and token management. It provides endpoints for creating and retrieving tokens, as well as admin login functionality.

## Access swagger UI
To access the swagger UI, run the application and navigate to:
```
root_url/swagger/index.html
```

## Update swagger
To update the swagger documentation, run the following command from the /src directory of the project:

```bash
swag init 
```
